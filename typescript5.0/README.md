# Typescript 5.0

엄청나게 많은 변화가 생긴 Typescript 5.0에 대해 알아보자.

## Decorators

데코레이터에 대한 지원이 공식적으로 추가되었다. (더이상 시험 기능이 아니다.)\
ECMAScript 빌트인 기능으로 추가되고 있으며, 이전과 달리 `experimentalDecorators` 옵션을 사용하지 않아도 된다.\
데코레이터를 통해 클래스, 메소드, 멤버 변수 등에 재사용 가능한 로직을 적용할 수 있다.
